Upgrading Shipping Guide

This guide shows you the difference between the versions of our Shipping Guide API, both the JSON/XML and SOAP interface.

We recommend always updating to the latest version and to do it at your earliest convenience. Upgrading to the latest version miximizes the time between upgrades.

The older 1.x versions will become deprecated and shut down some time in the future.
