Shipping Guide API schema 8 will be decomissioned 01.06.2020:

A new and improved version of our Shipping Guide API was launched in March 2018, and since then we have continuously phased out the old schemas. Version 1, 2, 3, 4, 5, 6, 7, 9 and 10 have already been phased out. On 01.06.2020, version 8 will be phased out. Make sure you have upgraded to the latest version of our Shipping Guide API to make sure your integration doesn't break. Go to Shipping Guide API 2.0 for more information.

The Pickup Point API becomes authenticated from 01.11.2020

Our customers are increasingly demanding new functionality and solutions that are easy to use. To meet these expectations, we must know which customers and partners are using our APIs and how they are using them. To further develop the value proposition of our Pickup Point API, the solution becomes authenticated from 01.11.2020. We recommend making the adjustment already now, well in advance of summer vacations and peak season. Go to Pickup Point API for more information.
